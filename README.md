## 📖 Deskripsi

**GUI FLY BY FIKZXX** adalah script fly premium untuk Roblox dengan antarmuka modern yang dioptimalkan untuk perangkat mobile. Dengan kontrol yang mulus, kecepatan yang dapat disesuaikan, dan sistem warna dinamis, script ini memberikan pengalaman terbang terbaik di Roblox.

## ✨ Fitur

### 🎮 Kontrol
- **Keyboard**: W/A/S/D untuk bergerak, Space untuk naik, Shift untuk turun
- **Mobile**: Swipe untuk bergerak, kontrol intuitif
- **Drag & Drop**: Geser GUI dengan jari

### 🎨 Tampilan
- **UI Modern**: Desain elegan dengan efek glow
- **Warna Dinamis**: Ubah warna tema dengan satu klik
- **Responsif**: Optimal untuk semua ukuran layar

### ⚙️ Kustomisasi
- **Slider Kecepatan**: Atur kecepatan 1-100
- **Config File**: Edit semua pengaturan di config.lua
- **Modular**: Mudah diintegrasikan dengan script lain
